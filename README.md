Pizzas

Overview
This project is an Android application that displays pizzas.

Technologies
- Kotlin
- Android SDK (minSdk 28, compileSdk 37)
- Jetpack Compose (Material3, Compose BOM)
- Gradle Kotlin DSL (build.gradle.kts)
- Koin (Dependency Injection)
- Retrofit + OkHttp (HTTP client)
- kotlinx.serialization (JSON)
- Coil (image loading)

Notes
Dependencies are managed via the version catalog (libs).
